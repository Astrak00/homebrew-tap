# Astrak00 Tap

## How do I install these formulae?

`brew install astrak00/tap/<formula>`

Or `brew tap astrak00/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "astrak00/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
